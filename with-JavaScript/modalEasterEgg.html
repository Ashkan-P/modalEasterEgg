This example shows you how to use modalEasterEgg.
This is the version which use the pure JavaScript.


modalEasterEgg uses both mouse and keyboard events. In the current page with your mouse go over this gray <div> and press the "i" from your keyboard; Now you can see the secret message.

All you need to do is:
  • Change the settings in modalEasterEgg.json:
    • theElement → The ID of the HTML element for the mouse event.
    • theMessage → The text of your message (You can add HTML tags too).
    • theCharacter → The character for keyboard event.
    • messageStyle → The CSS style for your message.
    • modalBlock → This is the modal block. Don't change it.
  • Add modalEasterEgg.min.js and modalEasterEgg.min.css to your codes.
  • Add <div id="the-div"></div> to the body of your webpages.
  • Put the modalEasterEgg.json file in the proper address (default is the same folder of your webpages).
